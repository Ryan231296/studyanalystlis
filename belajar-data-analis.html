Roadmap Data Analis

Jalur belajar terstruktur dari fundamental hingga expert. Estimasi total 6–12 bulan belajar konsisten.

📊
FASE 01
Fondasi & Mindset Data
Pahami apa itu data analisis, peran seorang data analis, dan cara berpikir berbasis data. Bangun fondasi yang kuat sebelum masuk ke teknis.
Data Literacy Statistik Dasar Jenis Data Data vs Information Problem Framing
⏱ 2–3 Minggu | Tingkat: Pemula
🐍
FASE 02
Python untuk Analisis Data
Kuasai Python sebagai bahasa utama data analis. Fokus pada library pandas, NumPy, dan manipulasi data. Tidak perlu mahir coding dulu — cukup bisa menganalisis data.
Python Basics Pandas NumPy Data Wrangling Jupyter Notebook File Handling (CSV, Excel)
⏱ 4–6 Minggu | Tingkat: Pemula–Menengah
🗄️
FASE 03
SQL & Database
SQL adalah skill wajib data analis. Pelajari cara query database, filter data, agregasi, join tabel, dan optimasi query. Hampir semua perusahaan menggunakan SQL.
SELECT, WHERE, GROUP BY JOIN (INNER, LEFT, RIGHT) Subquery & CTE Window Functions PostgreSQL / MySQL BigQuery / Snowflake
⏱ 3–4 Minggu | Tingkat: Pemula–Menengah
📈
FASE 04
Visualisasi Data & Storytelling
Data yang baik harus bisa dikomunikasikan dengan baik. Pelajari cara membuat visualisasi yang efektif dan menceritakan insight dari data kepada stakeholder non-teknis.
Matplotlib & Seaborn Plotly Tableau / Power BI Dashboard Design Data Storytelling Chart Selection
⏱ 3–4 Minggu | Tingkat: Menengah
📐
FASE 05
Statistik & Analitik Lanjutan
Dalami konsep statistik yang digunakan dalam analisis bisnis nyata. Dari descriptive analytics hingga predictive analytics dasar.
Descriptive Statistics Hypothesis Testing A/B Testing Regresi Linear Korelasi & Kausalitas Probabilitas
⏱ 4–5 Minggu | Tingkat: Menengah–Lanjutan
🏗️
FASE 06
Portofolio & Proyek Nyata
Bangun portofolio yang kuat dengan mengerjakan proyek analisis data end-to-end. Gunakan dataset nyata dari Kaggle, Google Dataset Search, atau data publik pemerintah.
GitHub Portfolio Kaggle Projects Case Study Bisnis EDA (Exploratory Data Analysis) Laporan Analisis Presentasi Temuan
⏱ 4–6 Minggu | Tingkat: Menengah–Lanjutan
🎯
FASE 07
Persiapan Karir & Sertifikasi
Siapkan diri untuk dunia kerja: CV data analis, persiapan interview teknis, negosiasi gaji, dan pilihan sertifikasi yang diakui industri.
CV & LinkedIn Optimization Interview Prep Google Data Analytics Cert IBM Data Analyst Cert Microsoft Power BI Cert Salary Negotiation
⏱ 3–4 Minggu | Tingkat: Semua Level
💡
Pro Tip: Jangan terlalu lama di satu fase. Konsistensi 1–2 jam per hari lebih efektif daripada belajar marathon di akhir pekan. Target selesai roadmap ini dalam 6–9 bulan dengan belajar rutin.
